Elementary Cellular Automaton Simulator

Rules

What's going on?

A small elementary cellular automaton simulator I built with Rust and WebAssembly. I wanted an introductory project to Rust, and this seemed a bit more fun and useful than a CLI to-do app. Using Rust makes it quite a bit faster than what you'd otherwise get.

How to use?

If you're already familiar with elementary cellular automaton, you can jump right in: select or create a rule, pick a starting condition, set the grid size, and press Generate to see how the cells evolve. Scroll down to see the generation if you're on mobile. Otherwise, Wolfram Mathworld provides a nice introduction.
